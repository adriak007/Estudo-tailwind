# 11 - Desafios Praticos

## Como usar este arquivo

Escolha um desafio por vez.
Nao copie pronto. Primeiro tente sozinho.

## Desafio 1 - Card de perfil

Monte:

- avatar
- nome
- cargo
- bio curta
- botao

Foco:

- espacamento
- borda
- sombra

## Desafio 2 - Hero section

Monte:

- titulo forte
- subtitulo
- dois botoes
- imagem ou bloco visual

Foco:

- hierarquia visual
- largura maxima
- responsividade

## Desafio 3 - Grid de servicos

Monte 6 cards em grid com:

- icone
- titulo
- texto

Foco:

- `grid`
- `gap`
- responsividade

## Desafio 4 - Formulario de login

Monte:

- campo de email
- campo de senha
- link de recuperar senha
- botao de entrar
- feedback visual de foco e erro

Foco:

- `focus-visible`
- `peer`
- acessibilidade basica

## Desafio 5 - Dashboard simples

Monte:

- sidebar
- header
- cards de metricas
- tabela

Foco:

- composicao de layout
- consistencia visual
- reuso de card

## Desafio 6 - Lista filtravel

Monte:

- input de busca
- tabs de filtro
- lista de itens
- estado ativo nas tabs

Foco:

- estados
- componentes repetiveis
- `aria-current` ou `data-state`

## Desafio 7 - Pagina de preco

Monte:

- tres planos
- badge de destaque no plano principal
- lista de beneficios
- CTA

Foco:

- comparacao visual
- destaque de um card
- alinhamento de informacao

## Desafio 8 - Navegacao mobile + desktop

Monte:

- menu mobile colapsado
- menu desktop horizontal
- item ativo

Foco:

- responsividade
- estados
- legibilidade de estrutura

## Criterios para se avaliar

- o mobile ficou bom antes do desktop?
- as classes estao legiveis?
- o espacamento esta coerente?
- repeticoes foram extraidas?
- ha foco visivel?

## Nivel extra

Depois de terminar qualquer desafio:

1. refaca sem olhar
2. troque a identidade visual
3. transforme em componente reutilizavel
4. adapte para React
